# Project Manager

Manage projects in Obsidian: track status, milestones, tasks, and progress with a kanban-style sidebar view.

## Features

- **New Project** — name, status, due date, priority, description
- **Add Milestone** — attach milestones to any project with a due date
- **Log Update** — record a progress update and set % complete
- **Kanban Sidebar** — columns by status (Planning / Active / On Hold / Complete), cards show priority badge, milestone count, and a progress bar
- **Archive Project** — move a project note into an Archive subfolder
- **Settings** — configure the projects folder

## Usage

Open the **Project Manager** panel from the ribbon icon. Use the command palette (`Ctrl/Cmd+P`) to:

- `Project Manager: New Project`
- `Project Manager: Add Milestone`
- `Project Manager: Log Update`
- `Project Manager: Archive Project`
